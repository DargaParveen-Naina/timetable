1. What is Python?

Python is a high-level, interpreted programming language known for its simple syntax and readability. It is used in web development, data science, AI, automation, and more.

2. Who developed Python and when?

Python was developed by Guido van Rossum and was first released in 1991.

3. What are the features of Python?

  • Easy to read and write
  • Interpreted language
  • Dynamically typed
  • Free and open-source
  • Cross-platform
  • Large standard library

6. What is the difference between Python 2 and Python 3?

Python 3 is the latest version and is not backward-compatible 2. Python 3 supports modern features like print() as a function and better Unicode handling.

7. What is indentation in Python?

Indentation is the space used at the beginning of a line to define blocks of code. Python uses indentation instead of braces {}.

8. What is the extension of Python files?

Python files have the extension .py.

9. What is an interpreted language?

An interpreted language executes code line by line instead of compiling the whole program at once. Python is an interpreted language.
